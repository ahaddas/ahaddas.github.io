Data Source Configuration

CSV Sources

Paste any publicly accessible CSV URL. Google Sheets: File → Share → Publish to web → CSV.

Promo Video

Optional. Link to a publicly accessible video file (MP4 recommended). Leave blank to skip video playback entirely.

⚠ Google Drive links don't work for video — Drive doesn't serve files directly to video players. Use Dropbox (change ?dl=0 → ?raw=1), GitHub raw, or any direct-serve host instead.

Logo / Background Ribbons

Optional. URL of an image to use as the repeating ribbon pattern on the sides of the display. Leave blank to hide ribbons.

Typography & Layout

Choose a display font for the kiosk. Select "Custom" to enter your own Google Fonts URL and family name.

Paste a Google Fonts stylesheet URL, then enter the font family name exactly as listed on Google Fonts.

Appearance

Customize colors and background. Leave background image blank to use the solid color.

Schedule Column Names

Enter the exact header names from your CSV (case-sensitive).

Leaderboard Column Names

For a single full-name column, fill Full name column and leave First/Last blank. For split columns, leave Full name blank and fill First + Last.
